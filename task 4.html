VSCODE

Visual Studio Code (VS Code) is a lightweight, powerful source code editor that's available for Windows, macOS, and Linux

FEATURES

Includes built-in support for JavaScript, TypeScript, and Node.js, plus a rich ecosystem of extensions for other languages and runtimes. It also has features like
  • Syntax highlighting
  • Bracket-matching
  • Auto-indentation
  • IntelliSense code completion
  • Debugging
  • Code refactoring
  • Support for Git
EMMET SYMBOL DESCRIPTION
> Nests elements inside each other
+ Places elements near each other, on the same level
^ Climbs one level up the tree
. Adds a class to an element
# Adds an ID to an element
: Adds attributes to an element
{} Adds text to an element
!: Adds a comment to an element
* Creates multiple elements

SHORTCUT KEYS

Keyboard shortcut keys are special keys on a keyboard that allow users to perform tasks more quickly.

keys DESCRIPTION
Ctrl + S To save the current file
Ctrl + O To open the selected file
Ctrl + C To copy the selected text
Ctrl + v To paste the selected text
Ctrl + x To cut the text
Ctrl + z To return the previous text
Ctrl + A To select all content file
Ctrl + Shift + p To open command palette
Ctrl + Shift + N To open new window
